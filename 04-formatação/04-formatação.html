HTML5

Elementos de formatação HTML

Negrito <b> e <strong>

O elemento <b> </b> define texto em negrito, sem qualquer importância extra

O elemento <strong> </strong> define texto de grande importância. O conteúdo interno normalmente é exibido em negrito

  • Exemplos: Texto em negrito
  • Exemplos: Texto importante

Itálico <i> e <em>

O elemento HTML <i> </i> define uma parte do texto com uma voz ou humor alternativo.
O conteúdo interno normalmente é exibido em itálico

Dica: A tag <i> costuma ser usada para indicar um termo técnico, uma frase de outro idioma, um pensamento

O elemento <em> </em> define o texto enfatizado. O conteúdo interno normalmente é exibido em itálico

Dica: um leitor de tela pronunciará as palavras com <em> ênfase, usando ênfase verbal

  • Texto em itálico
  • Texto enfatizado

Dica: Embora os resultados visuais sejam parecidos, é importante utilizar as tags semânticas corretas, assim os mecanismos de busca, leitores de tela etc. Tratarão os conteúdos com melhor eficiência
