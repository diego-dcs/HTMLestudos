Neste capítulo, examinaremos os elementos de cotação 

<blockquote cite="http://link.com"> 
O elemento <blockquote> define uma seção citada de outra fonte.

Os navegadores geralmente recuam elementos <blockquote>

<q> 
A tag <q> define uma citação curta.

Os navegadores normalmente inserem aspas ao redor da citação

<abbr tittle=""> 
A tag HTML <abbr> define uma abreviatura ou um acrônimo, como "HTML", "CSS", "Mr.", "Dr.".

A marcação de abreviaturas pode fornecer informações úteis aos navegadores, sistemas de tradução e motores de busca.

Dica: Use o atributo global title para mostrar a descrição da abreviatura/acrônimo ao passar o mouse sobre o elemento

<address> 
A tag <address> define as informações de contato do autor/proprietário de um documento ou artigo.

As informações de contato podem ser endereço de e-mail, URL, endereço físico, número de telefone, identificador de mídia social, etc.

O texto no elemento <address> geralmente é renderizado em itálico e os navegadores sempre adicionam uma quebra de linha antes e depois

<cite> 
A tag <cite> define o título de uma obra (por exemplo, um livro, um poema, uma música, um filme, uma pintura, uma escultura, etc.).

  Nota: O nome de uma pessoa não é o título de uma obra.

O texto no elemento <cite> geralmente é renderizado em itálico

<bdo> 
BDO significa Substituição Bidirecional.

A tag <bdo> é usada para substituir a direção atual do texto

Nota: Use o atributo de direção dir="" com um dos valoes

dir="rtl" right to left, direita pra esquerda
dir="ltr" left to right, esquerda pra direita
